Patient System Application
Click here for project timeline

The patient system application program in Java leverages Java's Graphical User Interface (GUI) capabilities to create an interactive platform for managing patient information. The application enables healthcare providers to easily input, update, and retrieve patient records through a user-friendly interface designed with the Java Swing interface. Key features include robust data validation, secure login for authorized access, and advanced search functionalities, ensuring efficient and accurate handling of patient data. By employing Java's GUI components, the application provides a seamless and visually appealing experience, facilitating basic and effective patient management in a healthcare setting. Lastly, the project is created to be an extended version of my previous Patient System project, but within a GUI setting.
