Client Server Chat Application

This program is a simple TCP-based client-server chat application written in C, where a client can connect to a server and exchange messages interactively. The server, implemented in server.c and server.h, initializes a socket, binds to a specified address and port (9001), and waits for client connections. When a client connects, the server enters a message exchange loop, allowing both sides to send messages until either sends "exit" or disconnects. The client, implemented in client.c and client.h, establishes a connection to the server and records both sent and received messages in a transcript.txt file. The program handles socket creation, connection management, message exchange, and cleanup, making it a straightforward example of TCP communication between a client and server.
