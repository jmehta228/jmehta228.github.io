Simple Data Pipeline

The Python program retrieves real-time weather data from the WeatherAPI, processes it, and stores it in a MySQL database using an ETL (Extract, Transform, Load) approach. First, the getAPIResponse function extracts data by sending a GET request to the API, returning the JSON response if successful. Next, getAPIInfo transforms this JSON data by parsing and organizing relevant information (such as local time, location, temperature, and wind speed) into a list format. Finally, the transferToDatabase function loads the transformed data into a MySQL database named "WeatherData" by executing an SQL insert command. The main function integrates these steps, constructs the API URL, and prompts the user for a city or zipcode to retrieve weather data. The Python program is modular, with clear separation of the ETL stages, enabling efficient data processing and easy maintenance.
