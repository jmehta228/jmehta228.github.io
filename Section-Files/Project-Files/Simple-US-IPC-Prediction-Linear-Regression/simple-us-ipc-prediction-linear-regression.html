Simple US Income per Capita Prediction with Linear Regression

This project uses a linear regression model to predict U.S. income per capita (IPC) based on historical data. After loading and exploring the dataset, the relationship between year and income is visualized, showing a strong positive correlation with a coefficient of 0.993. A simple linear regression model is then trained using the year as the independent variable and income as the dependent variable, enabling predictions like estimating IPC for the year 2023. Overall, the project demonstrates that U.S. income per capita has consistently increased over time and can be reliably forecasted using a linear approach.
