Brain Tumor Classification

This project applies machine learning techniques to classify brain tumors using MRI images. Leveraging a dataset from Kaggle with four tumor categories—glioma, meningioma, pituitary, and no tumor—it involves preprocessing the images (resizing and flattening), then training k-Nearest Neighbors and Decision Tree models. Model performance is evaluated through accuracy, precision, recall, and F1-score, with visual insights provided via confusion matrices and heatmaps.
